# Anime Rating Prediction using Regression

This project builds and evaluates multiple regression models to predict anime ratings using various independent variables.

## Overview

The goal of this project is to compare the performance of different regression models and determine how feature selection and hyperparameter tuning impact prediction accuracy.

## Models Used

- Baseline Regression Model
- Step Forward Feature Selection Regression
- Additional regression models (optimized with multiple configurations)

Each model was trained multiple times with different hyperparameter settings to improve performance and reduce overfitting.

## Evaluation Metrics

Models were evaluated using:

- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- R-Squared
- Adjusted R-Squared
- MAPE (Mean Absolute Percentage Error)

## Key Features

- Feature selection to reduce model complexity
- Multiple model comparisons
- Hyperparameter tuning across models
- Summary table and visualization of results
- Interpretation of model performance
